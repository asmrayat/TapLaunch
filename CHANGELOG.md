# Changelog

All notable changes to TapLaunch are documented here. Release downloads are available on [GitHub Releases](https://github.com/asmrayat/TapLaunch/releases).

## TapLaunch v1.0.0 — 2026-06-17

Initial public release.

- Double tap app launch
- Triple tap app launch
- Menu bar control
- Sensitivity settings (Low, Medium, High, and manual threshold)
- Tap test tool
- Launch at login support
- Local helper architecture for reliable motion sensor access on supported MacBooks
